# SimpleVectorClass
A simple C++ class for the Vector Data Structure

Some of the features include:
<ul>
<li>empty()</li>
<li>pop_back()</li>
<li>push_back()</li>
<li>clear()</li>
<li>set()</li>
<li>extend()</li>
<li>insert()</li>
<li>reverse()</li>
<li>print()</li>
<li>Overloaded [] operator</li>
</ul>
