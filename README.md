# SimpleVectorClass
A simple C++ class for the Vector Data Structure

Some of the features include:
<li>
<ul>empty()</ul>
<ul>pop_back()</ul>
<ul>push_back()</ul>
<ul>clear()</ul>
<ul>set()</ul>
<ul>extend()</ul>
<ul>insert()</ul>
<ul>reverse()</ul>
<ul>print()</ul>
<ul>Overloaded [] operator</ul>
</li>
